Breve descrição do sistema

O StockChef é um sistema inteligente de gerenciamento de estoque para restaurantes. Seu objetivo é auxiliar no controle de produtos armazenados, registrando entradas e saídas de itens, monitorando datas de validade e facilitando a organização do estoque.

O sistema também conta com recursos para leitura de QR Code ou código de barras para agilizar o cadastro de produtos e utiliza inteligência artificial para sugerir pratos com base nos ingredientes disponíveis no estoque, priorizando o uso de produtos próximos ao vencimento, reduzindo desperdícios e otimizando a gestão do restaurante.
